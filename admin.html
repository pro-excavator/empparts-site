Settings · 设置
Configure GitHub API connection and change admin password.
GitHub Connection · GitHub连接（必须先配置）
如何获取 GitHub Token（只需做一次）：
1. 打开 github.com → 右上角头像 → Settings
2. 左侧最底部 → Developer settings → Personal access tokens → Tokens (classic)
3. 点 Generate new token (classic)
4. Note 填 empparts-admin，Expiration 选 No expiration
5. 勾选 repo（第一个大项，全选）
6. 点底部 Generate token，立刻复制（只显示一次！）
