GitHub Connection · GitHub 连接设置
How to get a GitHub Token · 如何获取 Token：
1. Go to github.com → top-right avatar → Settings
2. Left sidebar → Developer settings → Personal access tokens → Tokens (classic)
3. Click Generate new token (classic)
4. Name it empparts-site, set expiration to No expiration
5. Check repo (full control of private repositories)
6. Click Generate token → copy it here immediately (shown only once!)
