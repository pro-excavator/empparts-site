🔗 GitHub Setup

配置 GitHub 信息，所有修改将直接写入你的仓库，所有设备自动同步。

GitHub Connection
Not tested
需要 repo 权限。在 GitHub → Settings → Developer settings → Personal access tokens 创建。
推荐直接放根目录 site-config.json，与 index.html 同级。
📖 使用说明

1. 填入 GitHub Token 和仓库信息，点击 Test Connection 验证

2. 在任意面板修改内容（Products、Gallery 等）

3. 点击顶部 保存到 GitHub — 修改写入 site-config.json

4. 网站 index.html 每次加载时从 GitHub 读取最新数据

5. ✅ 电脑、手机、平板打开网站，看到的内容完全一致
